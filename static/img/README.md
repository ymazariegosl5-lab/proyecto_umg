# Logo del Sistema

## Instrucciones

Para agregar el logo del sistema:

1. Coloque el archivo del logo con el nombre `logo.png` en esta carpeta (`static/img/logo.png`)
2. El logo se mostrará automáticamente en:
   - La página de inicio de sesión
   - La barra de navegación (cuando el usuario está logueado)

## Especificaciones recomendadas

- **Formato**: PNG (con transparencia) o JPG
- **Tamaño**: Máximo 200x200 píxeles para el login, 40px de altura para la navbar
- **Peso**: Menor a 500KB para mejor rendimiento

## Nota

Si el logo no está disponible, el sistema mostrará automáticamente un ícono de gota de agua como respaldo.
